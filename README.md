# TODOLIST
A simple TODOLIST web application built using HTML,CSS and Javascript.

## Features
-Add new tasks
-Mark tasks as completed
-undo completed tasks
-Filter tasks by status (All,completed,pending)
-Edit task name
-Delete tasks

# Technologies Used
-HTML
-CSS
-Javascript

# How to Run
1.Clone or download this repository.
2.Open todolist.html in your browser.
